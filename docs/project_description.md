# Descrição do Projeto

Este projeto tem como objetivo criar um script para provisionar automaticamente um servidor web Apache. A automação reduz o esforço manual e padroniza o processo, tornando-o mais rápido e confiável.

## Tecnologias Utilizadas

- Shell Script
- Apache HTTP Server
- Ubuntu Linux

## Funcionalidades do Script

1. Atualização de pacotes do sistema.
2. Instalação do servidor web Apache.
3. Criação de uma página inicial básica.
4. Verificação se o servidor está funcionando corretamente.
